# 写作指南

小红书母婴内容的写作规范和技巧。

---

## 📏 字数规范（必须遵守）

| 元素 | 字数要求 | 说明 |
|------|----------|------|
| **标题** | ≤ 20字 | 含emoji和标点 |
| **正文** | 约 1000字 | 800-1200字均可 |
| **标签** | 5-8个 | 不计入正文字数 |

---

## 写作原则

### 1. 真实感优先
- 像和闺蜜聊天一样自然
- 加入个人细节和经历
- 承认不完美和踩过的坑

### 2. 结构清晰
- 分段明确，每段一个重点
- 使用序号和符号辅助阅读
- 重要信息前置

### 3. 互动感强
- 多用"你"、"姐妹们"
- 结尾引导评论和收藏
- 提问式开头或结尾

---

## 文章结构模板

### 标准结构（约1000字）

```markdown
# [吸引人的标题，≤20字]

[开头段：痛点共鸣或问题引入，80-100字]

## 一、[核心内容1]
[详细内容，200-250字]
[IMAGE_PLACEHOLDER_1: 描述]

## 二、[核心内容2]
[详细内容，200-250字]
[IMAGE_PLACEHOLDER_2: 描述]

## 三、[核心内容3]
[详细内容，200-250字]

## 💡 总结/Tips
[关键要点，100-150字]

---
[互动引导：提问或呼吁，30-50字]

#标签1 #标签2 #标签3 #标签4 #标签5
```

---

## 配图占位符规范

在需要配图的位置使用以下格式：

```markdown
[IMAGE_PLACEHOLDER_X: 简要描述]
```

其中：
- `X` 为序号（1, 2, 3...）
- 描述应包含：主体、情绪、风格

**示例：**
```markdown
[IMAGE_PLACEHOLDER_1: 疲惫但温暖的妈妈抱着夜醒的宝宝，温馨插画风格]
[IMAGE_PLACEHOLDER_2: 睡前仪式流程图，信息图表风格]
```

## 个人背景融入

如果 `writing-system/background/` 目录存在以下文件，应读取并融入：

### my-story.md
- 提取身份信息（新手/二胎妈妈）
- 提取宝宝信息（月龄、性别）
- 提取地区信息

### experience.md
- 提取相关经验（如写睡眠训练，提取睡眠相关经验）
- 融入真实细节

**融入方式：**
```markdown
❌ 不好：很多妈妈都会遇到这个问题
✅ 好：我家宝宝6个月的时候也是这样，夜醒能有七八次
```

## emoji 使用规范

### 推荐使用
| 类型 | emoji | 用途 |
|------|-------|------|
| 强调 | 💡✨ | 重点提示 |
| 警告 | ⚠️❌ | 注意事项 |
| 成功 | ✅✔️ | 正确做法 |
| 情感 | 🥺😭 | 共情表达 |
| 宝宝 | 👶🍼 | 主题相关 |

### 使用原则
- 每段最多2-3个
- 开头和重点位置使用
- 不要连续使用超过2个
- 语气词后可加：~、！

## 标签策略

### 必加标签（3-5个）
1. **主题标签**：睡眠训练、辅食添加
2. **月龄标签**：6月龄、新生儿
3. **身份标签**：新手妈妈、二胎妈妈
4. **内容类型**：育儿干货、经验分享
5. **平台热门**：母婴好物、育儿日常

### 格式
```markdown
#睡眠训练 #6月龄宝宝 #新手妈妈 #育儿干货 #母婴
```

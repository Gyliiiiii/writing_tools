# 选题生成指南

为小红书母婴内容生成吸引人的选题和标题。

---

## 选题原则

### 1. 痛点驱动
- 从妈妈们的真实痛点出发
- 夜醒、喂养困难、发育焦虑是常见话题
- 用"你是不是也..."引发共鸣

### 2. 解决方案导向
- 标题要暗示有解决方案
- "亲测有效"、"终于解决"等词汇增加点击

### 3. 数字和具体化
- "3个方法"、"第5天就见效"
- 具体月龄："6个月宝宝"

---

## 三种选题角度

### 角度 A：问题解决型
**适用于：** 痛点明确、有解决方案的内容

**标题公式：**
```
[痛点描述]？[解决方案暗示]｜[身份标签]
```

**示例：**
- 宝宝夜醒10次？亲测有效的睡眠训练方法｜新手妈妈必看
- 辅食不吃怎么办？这个方法让宝宝爱上吃饭｜8月龄实战

**大纲结构：**
1. 痛点引入（共情）
2. 转折点（发现方法）
3. 方法介绍
4. 实操步骤
5. 效果展示
6. 注意事项

---

### 角度 B：科普教育型
**适用于：** 知识科普、辟谣、专业内容

**标题公式：**
```
[常见误区]≠[真相]｜[权威背书]
```

**示例：**
- 睡眠训练≠让宝宝哭｜儿科医生教你6种温和方法
- 6个月就要补铁？儿科医生告诉你真相｜附含铁食物表

**大纲结构：**
1. 辟谣/引入
2. 正确概念解释
3. 分类/对比表格
4. 如何选择/应用
5. 权威来源引用
6. 总结要点

---

### 角度 C：经验分享型
**适用于：** 个人经历、真实记录、好物推荐

**标题公式：**
```
从[问题状态]到[理想状态]｜[我的经历/记录]
```

**示例：**
- 从夜醒8次到一觉到天亮｜我家宝宝的睡眠训练记录
- 纯母乳喂到1岁的秘诀｜职场妈妈的背奶经历

**大纲结构：**
1. 背景介绍（我的情况）
2. 问题阶段描述
3. 转折/尝试过程
4. 效果展示（时间线）
5. 现在的状态
6. 给姐妹的建议

---

## 标题优化技巧

### 增加点击率的元素
| 元素 | 示例 | 效果 |
|------|------|------|
| 数字 | "3个方法" | ⭐⭐⭐⭐⭐ |
| 月龄 | "6月龄" | ⭐⭐⭐⭐ |
| 身份 | "新手妈妈" | ⭐⭐⭐⭐ |
| 效果词 | "亲测有效" | ⭐⭐⭐⭐⭐ |
| 情绪词 | "崩溃" | ⭐⭐⭐ |

### 避免的元素
- ❌ 过于专业的术语
- ❌ 夸大其词（"神奇"、"秒睡"）
- ❌ 过长的标题（超过25字）
- ❌ 全大写或过多符号

---

## 输出格式

生成的选题报告应包含：

```markdown
## 🎯 选题方案

### 方案 A：[角度名称]
**标题**: [完整标题]
**角度**: [简述角度]
**预估效果**: ⭐⭐⭐⭐⭐
**适合文风**: [推荐文风]

**大纲**:
1. [段落1主题]
2. [段落2主题]
...

### 方案 B：...
### 方案 C：...
```

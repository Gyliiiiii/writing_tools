## 术语处理原则

1. **首次出现**：中文译名（英文原名/缩写）
2. **后续出现**：仅使用中文译名
3. **专有名词**：保留英文，括号注释
4. **通用术语**：直接使用中文

### 示例
```markdown
第一次：睡眠训练中常用的费伯法（Ferber method）是一种渐进式方法...
第二次：使用费伯法时，需要注意...
```

# de-ai-ification Skill - 去AI味处理

让AI生成的内容更自然、更真实、更有人味。

---

## 技能概述

**目的：** 消除AI生成内容的机械感，让文章更自然
**输入：** AI生成的文章草稿
**输出：** 去AI味报告 + 修改后的文章

---

## AI味特征

### 1. 开头套话

```markdown
❌ "在当今社会..."
❌ "随着科技的发展..."
❌ "众所周知..."
❌ "作为一个..."
❌ "首先，让我们来了解..."
```

### 2. 过渡套话

```markdown
❌ "接下来，我们来看看..."
❌ "值得注意的是..."
❌ "不仅如此..."
❌ "更重要的是..."
❌ "总的来说..."
```

### 3. 结尾套话

```markdown
❌ "综上所述..."
❌ "总而言之..."
❌ "希望这篇文章对您有所帮助"
❌ "如果您觉得有用..."
```

### 4. 语气问题

```markdown
❌ 过于正式：您、以及、因此、从而
❌ 过于平铺：每句话都是陈述句
❌ 缺少情感：没有语气词和表情
❌ 结构死板：1、2、3、4 机械排列
```

## 去AI味模式

### 模式1：删除套话

| 原文 | 处理 |
|------|------|
| "在当今社会，越来越多的父母..." | 直接删除开头，从具体内容开始 |
| "接下来，让我们来看看..." | 删除，直接进入内容 |
| "综上所述" | 改为具体总结或删除 |

### 模式2：口语化替换

| 原文 | 替换 |
|------|------|
| "因此" | "所以" |
| "以及" | "和"、"还有" |
| "您" | "你" |
| "进行" | 直接用动词 |
| "非常" | "真的"、"特别" |
| "这种方法的核心理念是" | "说白了就是" |

### 模式3：添加语气词

| 位置 | 添加 |
|------|------|
| 句尾 | 呢、呀、吧、嘛、哈 |
| 强调处 | 真的、确实、绝了 |
| 转折处 | 但是吧、不过呢 |

### 模式4：拆分长句

```markdown
❌ 原文：
睡眠训练是一种帮助婴儿学习自主入睡的方法，通过建立固定的睡前仪式和逐渐减少父母的干预，让宝宝能够在夜间醒来后自己重新入睡。

✅ 修改：
睡眠训练是什么呢？简单说就是帮宝宝学会自己睡觉。
怎么做呢？建立固定的睡前仪式，慢慢减少哄睡干预。
目标是让宝宝夜醒后能自己接觉。
```

### 模式5：添加个人细节

```markdown
❌ 原文：
坚持了一周后情况有所改善。

✅ 修改：
坚持到第5天的时候，那天晚上宝宝居然只醒了1次！
我当时激动得差点哭出来。
```

## 自然度评分

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 套话数量 | 25% | 0个=满分，每个扣5分 |
| 口语化程度 | 25% | 是否有自然对话感 |
| 情感表达 | 20% | 是否有真实情感 |
| 句子多样性 | 15% | 长短句搭配 |
| 个人细节 | 15% | 是否有具体经历 |

**评分等级：**
- 90-100：非常自然
- 80-89：基本自然
- 70-79：有AI感
- 60-69：明显AI感
- <60：典型AI文

## 执行流程

### 1. 扫描检测
```
扫描全文，标记：
- 开头套话
- 过渡套话
- 结尾套话
- 过于正式的表达
- 长难句
```

### 2. 逐项处理
```
按模式处理每个标记：
1. 删除套话
2. 口语化替换
3. 添加语气词
4. 拆分长句
5. 添加细节（结合个人背景）
```

### 3. 整体润色
```
通读全文：
- 检查语气一致性
- 确保内容完整
- 验证自然度评分
```

## 小红书特色调整

### emoji 补充
```markdown
❌ "效果很好"
✅ "效果真的绝了 ✨"

❌ "需要注意"
✅ "⚠️ 这里要注意"
```

### 语气增强
```markdown
❌ "我觉得这个方法不错"
✅ "这个方法真的太香了！"

❌ "可能会有些困难"
✅ "前两天确实有点难熬 🥺"
```

### 互动感
```markdown
❌ "以上就是我的分享"
✅ "姐妹们觉得呢？评论区唠唠~"
```

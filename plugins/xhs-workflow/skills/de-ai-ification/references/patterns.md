# 去AI味模式库

完整的去AI味替换模式和规则。

---

## 开头套话库

### 必删套话

| 套话 | 替换建议 |
|------|----------|
| 在当今社会 | 直接删除，从具体内容开始 |
| 随着科技的发展 | 删除 |
| 众所周知 | 删除或改为"其实" |
| 作为一个XX | 改为直接陈述身份 |
| 首先，让我们来了解 | 直接开始内容 |
| 大家好，我是 | 改为"姐妹们好！" |

### 替换示例

```markdown
❌ 在当今社会，越来越多的妈妈开始关注宝宝的睡眠问题...
✅ 宝宝睡眠这个坑，踩过的妈妈都懂 🥺

❌ 作为一个新手妈妈，我深知育儿的不易...
✅ 自从当了妈，睡过几个整觉？我反正没有...
```

## 过渡套话库

### 必删套话

| 套话 | 替换建议 |
|------|----------|
| 接下来，让我们来看看 | 直接进入内容 |
| 值得注意的是 | 改为"这里要说一下" |
| 不仅如此 | 改为"而且"或"还有" |
| 更重要的是 | 改为"最关键的" |
| 需要指出的是 | 删除或改为"其实" |
| 在这里，我想强调 | 直接强调 |

### 替换示例

```markdown
❌ 接下来，让我们来看看具体的方法...
✅ 具体怎么做呢？

❌ 值得注意的是，这个方法需要坚持...
✅ 说真的，这个方法要坚持才有用
```

## 结尾套话库

### 必删套话

| 套话 | 替换建议 |
|------|----------|
| 综上所述 | 改为"总结一下"或直接总结 |
| 总而言之 | 改为"说白了就是" |
| 希望这篇文章对您有所帮助 | 改为互动引导 |
| 如果您觉得有用 | 改为"觉得有用的话" |
| 欢迎在评论区留言 | 改为"评论区唠唠呗" |

### 替换示例

```markdown
❌ 综上所述，睡眠训练是一个需要耐心的过程...
✅ 总之，坚持最重要！前几天难熬但挺过去就好了 💪

❌ 希望这篇文章对您有所帮助，欢迎点赞收藏
✅ 姐妹们有问题评论区问我，一起加油~
```

## 正式表达替换

| 正式表达 | 口语替换 |
|----------|----------|
| 因此 | 所以 |
| 以及 | 和、还有 |
| 您 | 你 |
| 进行 | 直接用动词 |
| 非常 | 真的、特别、超级 |
| 较为 | 比较、挺 |
| 此外 | 另外、还有 |
| 由于 | 因为 |
| 关于 | 说到、讲到 |
| 目前 | 现在 |
| 能够 | 能、可以 |
| 需要 | 要 |
| 但是 | 不过 |
| 然而 | 但是吧 |
| 总的来说 | 总之 |
| 这种方法的核心理念是 | 说白了就是 |
| 经过研究发现 | 研究表明 |
| 在实际操作中 | 实际上 |

## 语气词添加规则

### 句尾语气词

| 场景 | 可添加 | 示例 |
|------|--------|------|
| 陈述 | 呢、呀 | "这个方法挺好用的呀" |
| 建议 | 吧 | "你可以试试吧" |
| 强调 | 啊、哇 | "真的太有效了啊" |
| 疑问 | 吗、呢 | "你们也遇到过吗" |

### 语气强调

| 原表达 | 加强后 |
|--------|--------|
| 很好 | 真的很好 |
| 有效 | 超级有效 |
| 推荐 | 强烈推荐 |
| 不错 | 还挺不错的 |

## 长句拆分规则

### 拆分原则
- 超过30字的句子考虑拆分
- 一句话只说一件事
- 用短句增加节奏感

### 拆分示例

```markdown
❌ 睡眠训练是一种帮助婴儿学习自主入睡的方法，通过建立固定的睡前仪式和逐渐减少父母的干预，让宝宝能够在夜间醒来后自己重新入睡。

✅ 睡眠训练是什么呢？
简单说就是帮宝宝学会自己睡觉。
怎么做呢？
建立固定的睡前仪式，慢慢减少哄睡干预。
目标是让宝宝夜醒后能自己接觉。
```

## 个人细节添加

### 添加位置
- 描述问题时：加入自己的具体感受
- 描述效果时：加入时间线和具体变化
- 描述建议时：加入自己的尝试经历

### 添加示例

```markdown
❌ 坚持了一周后情况有所改善。
✅ 坚持到第5天的时候，那天晚上宝宝居然只醒了1次！我当时激动得差点哭出来。

❌ 这个方法需要耐心。
✅ 说实话，前两天我差点放弃了，但老公一直鼓励我坚持，第三天真的开始见效了。
```

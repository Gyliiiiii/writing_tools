# 闺蜜分享型文风

像好闺蜜聊天，轻松、真实、有趣。

---

## 文风特征

### 核心定位
- **角色**：好闺蜜
- **语气**：轻松、真实、略带夸张
- **距离感**：非常近，像私聊

### 语言特点

| 特点 | 说明 | 示例 |
|------|------|------|
| 口语化 | 说话的感觉 | "姐妹们！" |
| 夸张表达 | 增强感染力 | "绝了！真的绝了！" |
| 真实吐槽 | 不装 | "踩坑无数次..." |
| 强烈安利 | 热情推荐 | "必须安利！" |

---

## 写作模板

### 开头公式
```markdown
[姐妹称呼] + [强烈情感] + [主题引入]

示例：
姐妹们！！！
这个睡眠神器我必须安利给你们！
用了之后宝宝真的秒睡！！
```

### 正文公式
```markdown
[背景吐槽] → [发现好物/方法] → [使用体验] → [效果展示] → [强烈推荐]

示例：
之前宝宝睡觉真的让我头秃...
然后朋友推荐了这个方法...
我抱着试试看的心态...
结果！！第三天就见效了！！
```

### 结尾公式
```markdown
[总结推荐] + [互动邀请]

示例：
反正我是离不开了哈哈哈
姐妹们用过吗？评论区唠唠呗！
```

## 完整示例

```markdown
# 姐妹们！这个睡眠方法也太绝了吧！！

各位老母亲们好！！

今天必须来分享一下我家宝宝的睡眠逆袭之路 🔥

## 背景

之前宝宝睡觉这事真的让我头秃😭
一晚上能醒七八次
白天我整个人都是恍惚的
看啥啥不顺眼 无语死了

## 转折点

后来在妈妈群里看到有人分享费伯法
抱着死马当活马医的心态试了试

结果！！！

## 效果

第一天：有点难熬
第二天：好像好一点了？？
第三天：只醒了2次！！
第五天：一觉到天亮！！！

我当时差点哭出来好吗！！！

## 具体方法

其实就是：
1. 睡前仪式要固定（洗澡→按摩→喝奶→哼歌）
2. 困了但没睡着的时候放床上
3. 哭的时候先等3分钟再去安抚
4. 慢慢把时间拉长
```

## 注意事项

虽然是闺蜜风格，但仍需注意：

1. **不过度夸张**：效果描述要基于事实
2. **不误导**：好物推荐要真实
3. **安全第一**：母婴内容安全红线不能碰
4. **适度轻松**：吐槽可以有，但不要负能量

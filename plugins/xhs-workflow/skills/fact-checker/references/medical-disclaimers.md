# 医学声明模板

在涉及医学建议时使用的标准声明。

---

## 通用声明

### 开头声明
```markdown
💡 以下内容仅供参考，不能替代专业医疗建议。如有疑问，请咨询儿科医生。
```

### 结尾声明
```markdown
⚠️ 本文内容仅供参考，每个宝宝情况不同，具体问题请咨询专业医生。
```

---

## 分类声明

### 睡眠相关
```markdown
💡 睡眠训练方法因宝宝而异，开始前建议咨询儿科医生确认宝宝是否适合。
```

### 喂养相关
```markdown
⚠️ 辅食添加时间和方法请遵循儿科医生建议，尤其是有过敏家族史的宝宝。
```

### 发育相关
```markdown
💡 发育里程碑仅供参考，每个宝宝发育节奏不同。如有担忧请咨询儿保医生。
```

### 疾病护理
```markdown
⚠️ 本文仅供日常护理参考，如宝宝出现异常症状请及时就医。
```

---

## 就医建议模板

### 紧急情况
```markdown
🚨 以下情况请立即就医：
- 3个月以下宝宝发烧
- 呼吸困难
- 嗜睡、反应差
- 持续呕吐或腹泻
- 抽搐
```

### 需要关注
```markdown
⚠️ 以下情况建议咨询医生：
- 持续X天的症状
- 症状反复出现
- 影响正常饮食/睡眠
- 家长感觉不对劲
```

---

## 药物相关

### 禁止建议药物
```markdown
❌ 不要：自行给宝宝使用任何药物
✅ 应该：咨询儿科医生后遵医嘱用药
```

### 常见问题
```markdown
Q: 可以给宝宝吃XXX吗？
A: 任何药物或保健品的使用，请先咨询儿科医生。
```

---

## 使用位置

| 声明类型 | 使用位置 |
|----------|----------|
| 通用开头声明 | 涉及健康话题的文章开头 |
| 通用结尾声明 | 所有母婴文章结尾 |
| 分类声明 | 对应分类内容中 |
| 就医建议 | 疾病护理、异常症状相关内容 |
| 药物声明 | 提及任何药物时 |
